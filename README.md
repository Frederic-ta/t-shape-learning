# Programme T-shape "Caméléon" 🦎

*Formation complète en intelligence sociale et adaptation humaine*

## Vue d'ensemble

Le programme T-shape "Caméléon" est une formation intensive de 15 semaines (75 jours) pour développer une intelligence sociale avancée. Chaque semaine explore un aspect différent de l'art de lire, comprendre et s'adapter aux autres humains.

**Format** : 5 fiches par semaine, 20 minutes de lecture par jour  
**Niveau** : Progressif (débutant → expert)  
**Objectif** : Devenir un caméléon social capable de s'adapter à n'importe quelle situation humaine

## 🎓 Comment utiliser ce programme

1. **Une semaine = un thème** : Chaque semaine se concentre sur un aspect spécifique
2. **Progression quotidienne** : Chaque jour construit sur le précédent
3. **Pratique immédiate** : Chaque fiche contient un exercice du jour
4. **20 min/jour** : Format optimisé pour l'apprentissage régulier
5. **Récap hebdomadaire** : Le jour 5 consolide les acquis

## 💡 Conseils d'apprentissage

- **Régularité** : Mieux vaut 20 min/jour que 2h une fois
- **Pratique** : Teste immédiatement ce que tu apprends
- **Observation** : Applique les concepts en situation réelle
- **Patience** : Certaines techniques demandent du temps à maîtriser
- **Éthique** : Utilise ces compétences de manière responsable
