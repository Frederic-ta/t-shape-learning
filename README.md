# 🦎 T-shape Learning

Apprentissage T-shape — une semaine, un sujet, des compétences transversales.

## Méthode
- **1 sujet par semaine**, 5 jours, ~20 min de lecture/jour
- Chaque leçon : concept, science, détails, exercice pratique, quiz
- Objectif : devenir un caméléon — large surface de compétences, profondeur stratégique
